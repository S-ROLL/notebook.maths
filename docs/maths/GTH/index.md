# Giải tích hàm

!!! tip
    - Xác định:
        - Cái **đã có**.
        - Cái cần **chứng minh**.
    
    - Từ đó $\rightarrow$ Cái cần **tìm**.

!!! note
    - Giả thuyết $\rightarrow$ Kết quả.
        - Giả thuyết $\rightarrow$ **đã có**.
        - Kết quả $\rightarrow$ **cần tìm**.
