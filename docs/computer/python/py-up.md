# Python upgraded step-by-step

https://docs.python.org/3/tutorial/venv.html

## Homebrew
```
brew update
```

```
brew upgrade python
```
!!! info
    ```
    python3 --version
    ```

## Venv
```
which python3
```

```
deactivate
```

```
rm -rf venv
```

```
/opt/homebrew/bin/python3 -m venv venv
```

```
source venv/bin/activate
```

!!! info
    ```
    python3 --version
    ```
