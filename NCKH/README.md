### Octave
- Checking Big-O
  - $$O(2^n)$$
```
n = 1:100;
big_o = 2 .^n;
for i = 1:rows(n)
  for j = 1:columns(n)
    disp(sprintf("2 ^ %d = %d", n(i, j), big_o(i, j)));
  end
end
```
